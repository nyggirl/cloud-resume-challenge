# Cloud Resume Challenge

## Overview

This project is my implementation of the Cloud Resume Challenge, built as part of my Cloud Engineer portfolio.

The goal is to build a production-style serverless resume website on AWS while applying Infrastructure as Code, CI/CD, security best practices, and cloud architecture principles.
